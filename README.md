# AI чат-бот приёмной комиссии ТИУ (Тюменского Индустриального Университета)

## Возможности агента

 - 📚 Поиск актуальной информации о приёмной компании 2026
 - 💾 Система памяти (запоминает информацию о пользователе и полезные факты)
 - 🔍 Web - поиск + парсинг Web страниц

## Запуск

**.env.example**
```text
YANDEX_CLOUD_FOLDER_ID="<YOUR FOLDER ID>"
YANDEX_CLOUD_API_KEY="<YOUR API KEY>"
EMBEDDINGS_API_BASE_URL="http://localhost:8001"
```

```bash
docker-compose up -d
```
